# 🎵 Spotify Clone Web Player

## 📌 Project Overview
This project is a **Spotify Web Player Clone** built using **HTML, CSS, and JavaScript**.  
It replicates the basic user interface and functionality of Spotify's web music player, allowing users to browse playlists and control music playback.

The goal of this project is to practice **front-end development concepts**, including layout design, responsive UI, DOM manipulation, and media control using JavaScript.

## 🧠 How the Player Works

The JavaScript file performs the following tasks:

1. Loads songs dynamically into the playlist.
2. Controls the audio player using JavaScript audio APIs.
3. Updates the seek bar based on song progress.
4. Allows navigation between songs using next and previous buttons.
5. Controls volume through a range slider.
